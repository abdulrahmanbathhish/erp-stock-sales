HOW IT WORKS:
================================================================================

1. Windows computer runs the server (START_SERVER.bat)
2. Server automatically detects the Windows IP address
3. All devices on the same Wi-Fi can connect automatically
4. No technical configuration needed - just double-click and run!


FOR END USERS (Windows):
================================================================================

FIRST TIME:
  1. Double-click SETUP.bat
  2. Wait for installation
  3. (Optional) Double-click CREATE_SHORTCUT.bat

EVERY TIME:
  1. Double-click START_SERVER.bat (or desktop shortcut)
  2. Look at the window for the IP address
  3. Share that IP with other devices on the same Wi-Fi


FOR OTHER DEVICES (Phones, Tablets, etc.):
================================================================================

1. Connect to the same Wi-Fi network as the Windows computer
2. Open any web browser
3. Enter the IP address shown in the Windows server window
4. That's it! The system loads automatically
